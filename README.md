
# WeatherApp

Приложение для отображения погоды, написанное на Swift, с использованием архитектуры **Clean + MVVM**.

## Особенности

- Получение актуальной погоды в Москве с автоматическим обновлением каждый час.    
- Возможность вводить название любого города и получать его текущую погоду.
- Работа с API через собственный клиент на основе `URLRequest`.
    
## Технологии и архитектура

- Язык: Swift
- Архитектура: Clean + MVVM
- Сетевые взаимодействия: `URLRequest`
- JSON-декодирование с `JSONDecoder` (`keyDecodingStrategy = .convertFromSnakeCase`)
    
## Используемый API

Приложение интегрировано с [weatherapi.com](https://www.weatherapi.com/) для получения данных о погоде.
